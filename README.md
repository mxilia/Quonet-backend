# Quonet
A post forum web application where user can create, delete and like/dislike posts. User can also modify their handler and bio. Supports moderation via admin dashboard.

## Repositories
 - [Frontend](https://github.com/mxilia/Quonet-frontend)
 - [Backend](https://github.com/mxilia/Quonet-backend)

## Technologies
 - **Frontend:** Next.js, Tailwind CSS, tanstack, Zustand, Zod
 - **Backend:** Go, Fiber, GORM, PostgreSQL, Redis
 - **Cloud & Tools:** AWS, Supabase, Docker
